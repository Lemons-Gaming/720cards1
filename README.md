# ציר הזמן של השיעור — משחק חינוכי

משחק חינוכי בעברית שבו מסדרים 12 קלפי "מהלך שיעור" לפי הסדר הכרונולוגי הנכון על ציר זמן נחשי (8:00–9:50).
הקלפים נופלים אחד-אחד, וגוררים כל קלף למשבצת הזמן המתאימה.

## הפעלה

המשחק הוא קובץ `index.html` יחיד (HTML + CSS + JavaScript מוטמעים), ללא תלות בשרת או ספריות חיצוניות — רק תיקיית `images/`.

- **מקומית:** פתחו את `index.html` בדפדפן. אם תמונות לא נטענות עקב הגבלות `file://`, הריצו שרת קטן:
  ```
  python -m http.server 8000
  ```
  וגלשו אל `http://localhost:8000/index.html`.
- **אונליין:** ניתן לפרסם דרך GitHub Pages (Settings → Pages → Branch: `main`).

## מבנה

- `index.html` — כל המשחק (לוגיקה, עיצוב, נכסים מוטמעים).
- `images/` — תמונות הקלפים, הרקע והודעת הניצחון.
